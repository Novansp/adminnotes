# Recommended Text-Based Tutorials & Documentation

These resources provide detailed, text-based instructions for legacy Linux services and commands commonly used in system administration.  
They are curated for compatibility with terminal-based browsers such as **Lynx** and **w3m**, which render simple HTML more effectively than modern JavaScript-heavy sites.

---

## 1. The Linux Documentation Project (TLDP) — Best for Lynx

This is a classical repository of Linux documentation. The **HOWTO** guides are written in simple HTML with minimal styling and no JavaScript, making them ideal for terminal browsers.

**Resources**

- DNS HOWTO  
  https://tldp.org/HOWTO/DNS-HOWTO.html

- FTP HOWTO  
  https://tldp.org/HOWTO/FTP.html

- Linux File Permissions (Intro to Linux Guide)  
  https://tldp.org/LDP/intro-linux/html/sect_03_04.html

---

## 2. Ubuntu Community Help Wiki

This wiki provides documentation specifically focused on Ubuntu configurations.  
Its semantic HTML structure makes navigation straightforward in text-based browsers.

**Resources**

- DNS Server (BIND9)  
  https://help.ubuntu.com/community/BIND9ServerHowto

- FTP Server (vsftpd)  
  https://help.ubuntu.com/community/vsftpd

- DHCP Server (isc-dhcp-server)  
  https://help.ubuntu.com/community/isc-dhcp-server

---

## 3. Linuxize

Linuxize is a modern tutorial site generated with static HTML.  
Despite its contemporary design, the pages remain largely readable in terminal browsers because the tutorial content does not depend on client-side JavaScript.

**Resources**

- `chmod` / `chown` Command Guide  
  https://linuxize.com/post/chmod-command-in-linux/

- User Management in Ubuntu  
  https://linuxize.com/post/how-to-create-and-delete-users-on-ubuntu-18-04/
